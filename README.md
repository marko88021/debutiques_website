# Debutiques Website

This is the official website for Debutiques - premium 3D furniture visualization studio.

## Features
- Multi-language support (9 languages)
- Responsive design
- 3D visualization showcase
- Calendly integration for scheduling
- Blog system with Supabase backend

## Deployment
- **Live Site**: https://debutiques.com
- **Auto-deploys** from main branch via Netlify
- **Easy updates**: Run `deploy.bat` after making changes
